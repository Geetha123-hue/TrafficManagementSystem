## Conversation Flow Tips

### For Users:
1. Start with general questions like "How does this work?"
2. Ask specific follow-ups based on the response
3. Navigate to specific dashboard sections for more details
4. Use the chatbot as a guide to understand features

### Common Question Patterns:
- "How does [feature] work?" → System explains the feature
- "Tell me about [topic]" → Provides information
- "What's [metric]?" → Explains specific metrics
- "Can you..." → Capabilities explanation
- "Help!" → Offers options for assistance

### Best Practices:
- Be specific with your questions
- Use natural language (no special commands needed)
- Ask follow-up questions for clarification
- Navigate to relevant dashboard sections for live data
